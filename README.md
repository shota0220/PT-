理学療法士マッチングサービス構想（企画・設計フェーズ）

概要
理学療法士として医療現場で働く中で感じた構造的課題をもとに、
ITで解決できる可能性を検討したマッチングサービスの企画案です。
本リポジトリは、エンジニア未経験の立場から
課題整理・要件定義・設計思考を重視して作成しています。

背景・課題
理学療法士は医師の指示下でしか働けず、裁量が極端に小さい
自費リハ・予防分野のニーズは高いが、制度上グレー
専門性の高いPTほど評価されにくい構造

解決したいこと
PTが専門性で選ばれる仕組みを作る
利用者が安心してPTを選べる環境を作る
法律・制度を踏まえた持続可能なモデルを設計する

想定ユーザー
慢性疾患・予防目的の高齢者
専門性を活かしたい理学療法士

サービス構成（案）
PTプロフィール（専門領域・経験）
利用者側の課題ヒアリング
マッチング・予約機能
